# QuizzesQuick

Лёгкое веб‑приложение для создания и прохождения викторин **без бэкенда** — все данные хранятся и шифруются в URL.

[![Лицензия](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## Описание

QuizzesQuick — это минималистичное веб‑приложение, позволяющее:
* создавать викторины прямо в браузере;
* шифровать данные викторины в URL;
* делиться викториной по ссылке — любой, у кого есть ссылка, может её пройти.

**Ключевая особенность:** приложение не требует сервера или базы данных. Все данные сохраняются в зашифрованном виде в URL, что делает его максимально лёгким и портативным.

## Как это работает

1. **Создание викторины:** пользователь заполняет форму с названием, темой, вопросами и ответами.
2. **Сохранение:** при нажатии «Сохранить викторину» приложение:
   * собирает все данные;
   * шифрует их (например, с использованием `btoa` и `encodeURIComponent`);
   * формирует уникальную ссылку.
3. **Деление:** пользователь копирует и отправляет ссылку другим.
4. **Прохождение:** получатель переходит по ссылке, приложение расшифровывает данные из URL и отображает викторину для прохождения.

## Особенности

* **Без бэкенда:** всё работает в браузере, не требуется сервер.
* **Портативность:** викторина существует в виде ссылки — её легко сохранить и переслать.
* **Мгновенный доступ:** для прохождения не нужна регистрация.
* **Лёгкость:** минимальный размер файлов, быстрая загрузка.
* **Адаптивный дизайн:** корректно отображается на ПК и мобильных устройствах.

## Технологии

* **HTML5** — структура приложения.
* **CSS3** — стильное оформление и адаптивный дизайн.
* **JavaScript (ES6+)** — логика создания, шифрования, расшифровки и прохождения викторин.
* **URL API** — работа с параметрами URL (`URLSearchParams`).
* **Web Cryptography API** (опционально) — для более надёжного шифрования.

## Функциональность

### Для создателя викторины

* ввод названия и темы викторины;
* добавление нескольких вопросов;
* для каждого вопроса: ввод текста, вариантов ответов, указание правильного ответа;
* кнопка «Сохранить викторину» — генерирует и отображает ссылку;
* возможность скопировать ссылку одним кликом.

### Для участника викторины

* просмотр названия и темы;
* последовательное прохождение вопросов с выбором ответа;
* мгновенная обратная связь (показывает правильный ответ);
* итоговый результат после завершения (количество правильных ответов).

## Использование

### Создание викторины

1. Перейдите на [страницу приложения](https://Rad414.github.io/quizzesquick).
2. Заполните поля:
   * **Название викторины** (например, «Викторина по JavaScript»);
   * **Тема** (например, «Программирование»);
   * добавьте вопросы и ответы, укажите правильные варианты.
3. Нажмите кнопку **«Сохранить викторину»**.
4. Скопируйте появившуюся ссылку.
5. Отправьте ссылку участникам.

### Прохождение викторины

1. Получите ссылку от создателя.
2. Перейдите по ней — викторина автоматически загрузится.
3. Отвечайте на вопросы.
4. Посмотрите итоговый результат.

## Безопасность и ограничения

* Данные шифруются, но **не защищены криптографически стойко** (используется базовое кодирование). Не используйте для конфиденциальной информации.
* Длина URL ограничена — слишком большая викторина может не поместиться в ссылку.
* Данные теряются, если очистить историю браузера или удалить ссылку.
